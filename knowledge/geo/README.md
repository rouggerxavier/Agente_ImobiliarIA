# Guia geografico (MVP)

Arquivos por local com estrutura fixa para melhorar busca semantica e filtros.

## Estrutura atual

- `joao_pessoa/`: cobertura total da lista operacional de bairros.
- `cabedelo/`: bairros prioritarios de Cabedelo.
- `bayeux/`: visao municipal inicial.
- `santa_rita/`: visao municipal inicial.

## Status atual

- Joao Pessoa: 63 arquivos de bairros + `README` local.
- Cabedelo: base inicial pronta.
- Bayeux: base inicial pronta.
- Santa Rita: base inicial pronta.
