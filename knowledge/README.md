# Knowledge Base (MVP)

Esta pasta guarda conteudo interno para respostas de QA sem inventar informacoes.

## Padrao de metadados

Use este bloco no topo de cada arquivo:

```yaml
---
domain: institutional | geo
city: joao_pessoa | cabedelo | bayeux | santa_rita | null
neighborhood: nome_do_bairro | null
topic: financiamento | custos | visita | condominio | temporada | glossario | bairro_guide
updated_at: 2026-03-06
---
```

## Regras editoriais

- Escrever em tom conservador: "em geral", "normalmente", "pode variar".
- Evitar percentuais e prazos exatos sem fonte oficial.
- Sempre incluir: "isso pode variar; confirmamos no caso concreto".
- Evitar promessa juridica, fiscal ou bancaria.
